# Projet-martketing-digital
Ce projet vise à analyser et croiser des données marketing et commerciales (leads, campagnes, CRM) afin d’évaluer la performance des canaux, comprendre le comportement des leads et identifier les profils clients les plus réactifs, en s’appuyant sur un périmètre maîtrisé et une sélection rigoureuse de variables pertinentes.
